# Shapo Digital — Agency Website

אתר הסוכנות של שאפו דיגיטל | מישל זוויגי

סוכנות פרסום דיגיטלית פרימיום — בניית אתרים בארכיטקטורת Serverless Edge, אוטומציות AI, ניהול קמפיינים, הפקת וידאו וקורסים.

## מבנה האתר (8 עמודים)
- `index.html` — דף הבית
- `services.html` — כל השירותים
- `web.html` — בניית אתרים
- `video.html` — הפקת וידאו
- `courses.html` — קורסים ומנטורינג
- `about.html` — אודות
- `contact.html` — צור קשר
- `portfolio.html` — תיק עבודות

## טכנולוגיה
HTML / CSS / JS סטטי — מוכן לדפלוי על Vercel ללא build.
